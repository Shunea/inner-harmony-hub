# OG Image pentru SEO

## Ce trebuie făcut:

1. Creați o imagine OG (Open Graph) pentru site cu următoarele specificații:
   - **Dimensiune**: 1200 x 630 px (raport 1.91:1)
   - **Format**: JPG sau PNG
   - **Nume fișier**: `og-image.jpg`
   - **Locație**: `/public/og-image.jpg`

## Conținut recomandat pentru imagine:

- Fotografia Nataliei Șargu
- Text: "Natalia Șargu"
- Subtitlu: "Psiholog & Psihoterapeut Integrativ"
- Locație: "Chișinău | Online"
- Culori brand: Maro (#8B7355), Bej (#F5F1ED)
- Logo sau simbol (dacă există)

## Exemple de text pe imagine:

```
Natalia Șargu
Psiholog & Psihoterapeut Integrativ

✓ Terapie pentru anxietate și depresie
✓ Cabinet în Chișinău
✓ Ședințe online pentru diasporă
```

## Verificare:

După creare, testați imaginea pe:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/

## Notă:

Imaginea trebuie să fie clară, profesională și să reprezinte brandingul site-ului.
Textul trebuie să fie lizibil și pe thumbnail-uri mici (ex: 300px width).
